# 대학 로고 이미지 폴더

각 레벨의 대학 로고 이미지입니다.
실제 대학 공식 로고(PNG/SVG)로 교체하려면 동일한 파일명으로 덮어쓰세요.

## 파일 목록

| 파일명 | 레벨 | 대표 대학 예시 | 누적 점수 |
|--------|------|--------------|---------|
| `level1.svg` (or `.png`) | 지방 전문대 | 부산경남대학 등 | 0 ~ 19점 |
| `level2.svg` (or `.png`) | 지방 4년제 | 부산대, 경북대 등 | 20 ~ 39점 |
| `level3.svg` (or `.png`) | 인서울 하위 | 국민대, 세종대 등 | 40 ~ 59점 |
| `level4.svg` (or `.png`) | 인서울 중위 | 한양대, 외대 등 | 60 ~ 79점 |
| `level5.svg` (or `.png`) | 인서울 상위 | 연세대, 성균관대 등 | 80 ~ 94점 |
| `level6.svg` (or `.png`) | 고려대학교 | 고려대학교 | 95 ~ 100점 |

## 교체 방법

1. 원하는 대학 로고 이미지를 준비 (PNG 투명배경 권장, 128×128px 이상)
2. 해당 레벨 번호로 파일명 변경 (예: `level6.png`)
3. 이 폴더에 붙여넣기 (덮어쓰기)
4. `src/App.tsx` 의 logo 경로에서 확장자를 `.png` 로 변경

> **주의**: 대학 공식 로고는 저작권·상표권이 있으므로 개인 학습 용도로만 사용하세요.
