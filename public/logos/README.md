# 대학 로고 이미지 폴더

각 레벨에 해당하는 대학 로고 이미지를 이 폴더에 넣어주세요.

## 파일명 규칙

| 파일명 | 레벨 | 누적 점수 |
|--------|------|---------|
| `level1.png` | 지방 전문대 | 0 ~ 19점 |
| `level2.png` | 지방 4년제 | 20 ~ 39점 |
| `level3.png` | 인서울 하위 | 40 ~ 59점 |
| `level4.png` | 인서울 중위 | 60 ~ 79점 |
| `level5.png` | 인서울 상위 | 80 ~ 94점 |
| `level6.png` | 고려대학교 | 95 ~ 100점 |

## 권장 사양

- 형식: PNG (투명 배경 권장)
- 크기: 128×128px 이상
- 파일 크기: 100KB 이하 권장

## 사용법

이미지를 이 폴더에 넣으면 `App.tsx`에서 `/logos/level1.png` 경로로 불러옵니다.
