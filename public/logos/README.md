# 대학 로고 이미지 폴더

각 레벨의 대표 대학 PNG 로고 파일입니다 (식품 관련 학과 보유 대학 기준).

## 파일 목록

| 파일명 | 레벨 | 대표 대학 (식품 관련 학과) |
|--------|------|-----------------------|
| `level1.png` | 지방 국립대 하위 | 충남대·전남대·경상국립대·강원대·제주대 |
| `level2.png` | 지방 국립대 상위 | 부산대·경북대·전북대·인하대·충북대 |
| `level3.png` | 인서울 하위 | 서울과기대·건국대·동국대·서울여대·숙명여대 |
| `level4.png` | 인서울 중위 | 경희대·중앙대·이화여대·한양대(에리카) |
| `level5.png` | 인서울 상위 | 한양대(서울)·연세대·서울대 |
| `level6.png` | 고려대학교 | 고려대학교 식품공학과 |

## 교체 방법
원하는 대학 공식 로고를 동일한 파일명(level1.png ~ level6.png)으로 교체하면 됩니다.
권장 사양: PNG 투명배경, 256×256px 이상
